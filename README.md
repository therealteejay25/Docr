# Docr

A software project for managing repositories.

## Features
- Repository management interface
- React-based frontend
- Next.js implementation

## Repository Page
The repository page (`app/repos/page.tsx`) provides an interface for viewing and managing repositories.

## Getting Started
1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`

## Technologies
- Next.js
- React
- TypeScript
